Add Channel Logger to your discord server!

Features

-Date and time (Year/Month/Day Hour:Minute:Second) of every message sent.

-The name of ther user who sent the message.

-The name of and url of every (if any) attachment in each message. (FileName FileURL)

-UTF-8 Logging, if a language is supported by UTF-8 it can be logged.

-Works with all channel types (DM, Group, Text)

Usage

"!SaveLog" Will start the channel log, once finished the bot will upload the channel log as a text file to the channel.

"!SaveLogPrivate" Will start the channel log, once finished the bot will direct message the initiator of the command with the text file that contains the channel log.
